笔记标题: HTML & CSS 基础

1. HTML 基础

HTML（HyperText Markup Language）是创建网页的标准标记语言。它由一系列元素组成，这些元素告诉浏览器如何显示内容。

2. CSS 基础

CSS（Cascading Style Sheets）用于控制网页的样式和布局。你可以使用CSS来调整颜色、字体、间距和布局等。

2.1 选择器

选择器用于选择你想要样式化的HTML元素。常见的选择器包括类选择器、ID选择器和元素选择器。

/* 选择器示例 */
p {
    color: blue;
    font-size: 16px;
}

2.2 盒模型

CSS盒模型描述了元素内容、内边距（padding）、边框（border）和外边距（margin）的结构。

图1: CSS 盒模型

3. JavaScript 基础

JavaScript 是一种轻量级的编程语言，它用于为网页添加动态行为，比如响应用户操作、修改内容等。

JavaScript is the programming language of the Web.
备注: JavaScript与Java并没有直接关系，虽然它们的名字相似。
